# blackhole-hvh
BLACKHOLE CLIENT — CS2 External HvH (Beta)

External DLL for Counter-Strike 2, mainly built for HvH but works for legit too. Inject with any injector, INSERT to open menu, END to unload.

Features:

Silent ragebot with per-weapon configs, engine-accurate hitchance (256 seeds via native CalcSpread + ClipRayToEntity), autowall, multipoint, auto scope, auto shoot, auto stop
Anti-aim (pitch/yaw modes, camera fix, movement fix, desync indicator)
Triggerbot with rage-aware mode
ESP (box, skeleton, health, name, distance, glow, bomb timer, spectator list)
Bullet tracers, third person camera
World visuals (night mode, skybox, fog, wall colors, flat shading, presets)
Skin changer with full paint kit selection, wear, seed, stattrak
Misc: bhop, auto strafe, no flash, radar hack, fov changer, rcs
Vote detector with animated notifications
Shot/miss HUD notifications
Known issues (beta):

Scout (SSG08) auto shoot doesn't work — tracks targets but won't fire
World visuals tab is inconsistent, scene hooks need rework
